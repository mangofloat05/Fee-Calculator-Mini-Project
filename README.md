# Barkota Fee Calculator

## Project Overview
A C# Windows Forms application designed to automate fee computations

## Architecture & Design Decisions
* **Platform**: Developed using C# and the .NET Framework.
* **Event-Driven Design**: Logic is triggered by user interactions (button clicks or Enter key) rather than a linear command-line flow.
* **What's in this codebase?**: The codebase is split into sections with respective functions:
    * **Section 5.1 (Upgrade Fee)**: Handles surcharge logic with a specific rounding requirement.
    * **Section 5.2 (AR Amount)**: Handles total balance calculations.
* **Error Handling**: Implemented `try-catch` blocks to prevent application crashes from invalid user inputs.

## How to Run
1. Open the project in **Visual Studio Community**.
2. Press **F5** or the **Start** button.
3. Enter the required amounts and press **Enter** or click **Calculate**.
